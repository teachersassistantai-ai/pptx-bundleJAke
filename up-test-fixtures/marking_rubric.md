# Marking Rubric - States of Matter Investigation Report

| Grade | Descriptor |
|-------|------------|
| A | Explains all changes of state precisely using the particle model; controls variables rigorously and justifies the choices; analyses reliability with insight and suggests valid improvements. |
| B | Explains changes of state using the particle model; identifies and controls the key variables; draws valid conclusions supported by the data. |
| C | Describes changes of state with some reference to the particle model; identifies the main variables; states a basic conclusion linked to the results. |
| D | Describes a change of state with limited accuracy; provides a partial method; conclusion is unclear or not linked to results. |
| E | Minimal, incomplete or off-task response. |

**Note to teachers:** Achievement standard alignment - AC9S7U01, AC9S7U02, and Science Inquiry strands (questioning and predicting, planning and conducting, processing and analysing, evaluating).

# Year 7 Science - Culminating Assessment
## States of Matter Investigation Report

**Task:** Students design and conduct a fair-test investigation into a change of state (for example melting or evaporation), then write a scientific report.

**Report sections required:**
1. Aim and hypothesis, framed using the particle model.
2. Method, with variables clearly identified (independent, dependent, controlled).
3. Results table and an appropriate graph.
4. Discussion that explains the results using the particle model.
5. Conclusion and an evaluation of the reliability of the investigation.

**Conditions:** Completed over 3 lessons - Lesson 1 planning, Lesson 2 conducting, Lesson 3 writing. Individual submission. Two A4 pages maximum.

**Submission:** Written scientific report (typed or handwritten) plus the completed results table.
